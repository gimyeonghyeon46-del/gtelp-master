## ✨ 특징

- 🎯 **게임화 학습** - 레벨업, 경험치, 생명 시스템
- 📚 **25개 문제** - 문법 15 + 어휘 10
- 🇰🇷 **완전 한국어** - UI 전체 한국어 지원
- 💯 **무료** - 누구나 무료로 사용 가능

## 🚀 기술 스택

- **Frontend:** Next.js 16, React 19, Tailwind CSS
- **Auth:** Clerk
- **Database:** Neon PostgreSQL
- **Deploy:** Vercel
